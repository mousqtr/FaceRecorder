# FaceRecorder

## Installer les librairies

1) Ouvrir un terminal et se rendre dans le dossier où se trouve le package.json
2) Taper la commande :  

        npm install

## Lancer le projet

1) Taper la commande :  
    
        npm start

2) Accéder à http://localhost:3000 via un navigateur
